# MyProfileApp

オリジナルの自己紹介用アプリになります。
本アプリで使用した技術・言語など、本アプリに関する内容から、私自身の学歴・職務経歴などを記載しております。

・使用言語
Laravel

・本番環境
AWS（EC2）

・データベース
AWS（RDS）
